# Rect Boilerplate

Minimal boilerplate, with sass loader, hot loader and dev server. Structually implemented Redux as well.

## Installation

1. Clone it
2. npm/yarn install

## Usage

Development: ``` npm run start ```    
Building: ``` npm run build ```    
It will output to ``` ./dist ```
